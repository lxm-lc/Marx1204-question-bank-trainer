# MARX1204 Question Bank Trainer

这是一个纯前端、本地运行的题库练习网页，用来把题库解析成可持续做题、可复盘、可导出报告的练习系统，帮助同学们进行马克思主义原理课程的复习。

项目围绕“本地题库练习”做了完整闭环：

- 自动解析题库文本并重编号
- 自动提取并隐藏题干中已有答案
- 按顺序单题提交作答
- 维护“不确定题库”和“做错了题库”
- 支持复盘、搜索、筛选、统计、导出
- 自动去重并兼容旧存档迁移
- 支持题目备注、错题导出、进度备份

## 功能概览

- 题库导入
  - 自动尝试读取根目录下的 `base.txt`
  - 也支持手动导入题库文本文件
  - 自动识别题干、选项、答案

- 题目解析规则
  - 以阿拉伯数字开头的行为题目题干
  - 以 `A/B/C/D/...` 开头的行为选项
  - 题库里残留的无关中文标题行会尽量忽略
  - 原始题号不要求连续，系统会重新生成连续编号

- 作答体验
  - 支持单选题和多选题
  - 每题单独提交，不允许空选提交
  - 提交后立即显示对错和正确答案
  - 提交后当前题选项锁定，不能再改
  - 支持复制题目文本；提交后复制内容会附带“我选的答案”和“正确答案”

- 错题 / 不确定题管理
  - 可把题目标注为“不确定”
  - 做错的题会自动进入“做错了题库”
  - 如果同一题既不确定又做错，只保留在“做错了题库”
  - 重新做对后，会自动从当前错题/不确定题库移出
  - 但“曾经做错 / 曾经不会”标签仍会保留，除非手动删除

- 复盘与筛选
  - 支持按连续编号跳转复盘
  - 支持关键词搜索题干、选项、原题号
  - 支持按未做、已做、当前错题、当前不确定、曾错、曾不会等条件筛选

- 统计与导出
  - 提供错题 / 不确定题重练统计页
  - 支持导出错题 / 不确定题为 Markdown
  - 支持导出错题 / 不确定题为 PDF 打印视图
  - PDF 导出为“题目 + 选项 + 正确答案”的打印格式

- 备注功能
  - 提交题目后可为该题添加备注
  - 复盘历史题目时也可多次修改备注
  - 以最后一次提交的备注为准
  - 空白备注视为未备注
  - 备注会跟随 Markdown / PDF 导出

- 进度保存与迁移
  - 进度保存在浏览器本地 `IndexedDB`
  - 支持导出 / 导入 JSON 进度备份
  - 支持“从头重做”主题库
  - 支持“清空所有进度”
  - 支持“重建题库”

- 重复题处理
  - 对题干、选项、答案完全一致的题自动去重
  - 当前仓库附带的 `base.txt` 会从原始解析 `704` 题去重为 `573` 题
  - 仍会保留重复题对应的全部原题号，便于追溯
  - 旧版本存档迁移规则：
    - 如果一组重复题里出现过错误，保留“最后一次做错”的记录
    - 如果一组重复题一直都做对，保留“第一次做对”的记录

## 技术栈

- HTML
- CSS
- 原生 JavaScript（ES Modules）
- IndexedDB
- Python 本地静态服务器
- Node.js 烟雾测试脚本

这是一个无后端依赖、无打包步骤、无 npm 依赖的本地网页项目。

## 正确启动方式

不要直接双击 `index.html`。

这个项目依赖：

- `type="module"` 的浏览器模块脚本
- `fetch("./base.txt")`
- 浏览器本地 `IndexedDB`

很多浏览器在 `file://` 方式直接打开页面时，会限制模块脚本或本地文件读取，表现通常是：

- 页面能打开，但按钮点击没反应
- 自动导入 `base.txt` 失败
- 控制台出现模块脚本 MIME 类型错误

### 启动方法

运行：

```powershell
python .\serve.py --host 127.0.0.1 --port 8000
```

然后在浏览器打开：

```text
http://127.0.0.1:8000/
```

### 停止服务

在启动服务器的终端里按 `Ctrl + C` 即可。

## 使用方式

1. 把题库文本放在项目根目录，并命名为 `base.txt`
2. 启动本地服务并打开网页
3. 页面会自动尝试导入 `base.txt`
4. 如果自动导入失败，可以手动选择题库文件
5. 之后即可按主题库顺序做题
6. 需要时可导出进度 JSON

## 数据保存说明

- 题目状态、错题记录、不确定记录、备注、复盘位置都保存在浏览器本地 `IndexedDB`
- 这些数据不会自动写入仓库，也不会被 Git 跟踪
- 换浏览器、换设备、清浏览器数据后，本地进度会丢失
- 如果要迁移进度，请主动使用“导出进度”功能保存 JSON 备份

## 测试

如果本地装了 Node.js，可以运行：

```powershell
node .\scripts\smoke-test.mjs
```

这个脚本会检查：

- 题库解析结果
- 去重结果
- 判题状态机的关键行为
